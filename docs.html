02Kurulum

HTML script tag

03Hizli Baslangic

// Tum ajanlari cek
ValorantFandom.getAgents().then(function(agents) {
  console.log(agents.length); // 29
  console.log(agents[0].name); // "Astra"
});

// Tek rank
ValorantFandom.getRank('Gold 2').then(function(r) {
  console.log(r.image_url); // raw URL
});
Cache: Ayni istek ikinci kez cekilmez, memory'de tutulur. clearCache() ile sifirlanir.

04Agents

});

07Ranks

25 rank ikonu (Iron → Radiant). Her rankin image_url'i GitHub raw olarak gelir — wikia CDN engellemesine takilmaz.

MetodDonus
getRanks()Tum ranklar (25)
getRanks('Gold')Gold 1/2/3
getRank('Gold 2')Tek rank (isim veya dosya adi)
ValorantFandom.getRanks('Diamond').then(function(ranks) {
  ranks.forEach(function(r) {
    img.src = r.image_url; // raw.githubusercontent.com/...
  });
});

08Skins

1068 skin, 19 silaha ait. getSkins() tumunu, silah adi verilince o silahin skinlerini doner.

// Tum skinler
ValorantFandom.getSkins().then(function(all) {
  console.log(Object.keys(all)); // ["Classic","Vandal",...]
});

// Sadece Vandal skinleri
ValorantFandom.getSkins('Vandal').then(function(skins) {
  console.log(skins.length); // 96
});
